Veja como Realizar filtro de Dados no PowerApps.

by Rudimar Althof

Filtrando galerias no Power Apps com a função Search, como funciona?

 

Para localizar registros em uma fonte de dados dentro do Power Apps podemos utilizar a função Search.

Ao usar a função Search, passamos uma substring (um pedaço de texto) que será checada se a mesma faz parte em cada uma das colunas informadas. Caso sim, estes serão incluídos no resultado.

Esta função retornará uma table que contém todos os registros encontrados, com as mesmas colunas da tabela original.

A função Search possui os seguintes parâmetros, todos obrigatórios:

  • Tabela: a fonte de dados / tabela com os dados que serão pesquisados;
  • SearchString: o trecho de texto a ser pesquisado. Caso este trecho seja branco ou vazio, a função retornará todos os registros;
  • Colunas: as colunas da tabela que serão pesquisadas. 
    • devem conter texto
    • devem estar entre aspas duplas
    • devem ser valores estáticos, não podendo ser calculados por uma fórmula

Nesta função, se a SearchString estiver contida dentro de qualquer uma das colunas passadas como argumento, o registro será retornado.

Importante: para Excel e SharePoint, os espaços em branco contidos nos nomes das colunas devem ser substituídos por "_x0020_". Exemplo: "Data de Inicio" fica "Data_x0020_de_x0020_Inicio"
